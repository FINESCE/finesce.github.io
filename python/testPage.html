DSE Description¶

WP1 intends to create a Domain Specific Enabler (DSE) which is an integration implementation kit consisting of a documented process including some examples for the established integrations within WP1. This will help others to easier and quicker establish those integrations. The kit will fit into the architecture as illustrated in enclosed image.

The kit will include a process to create flow of data from a surrounding environment into the set of GE’s that are included in the WP1 design. The data flow will be trough the Context Broker to and from both Big Data and Complex Event Management. It may also include how WP1 aims to implement the Access Control and Identity Management with data allocated within E.ON API Proxy environment. The documentation will include examples of potential difficulties that may arise during the implementation phase and how they have been solved.
